# Learning Java OOP

A personal repository created while learning and practicing **Object-Oriented Programming (OOP) in Java**.

This repository contains implementations, examples, and practice programs covering fundamental and advanced Java OOP concepts. The primary goal is to strengthen programming skills through hands-on coding and experimentation.

---

## Topics Covered

### Classes and Objects

* Creating classes and objects
* Constructors
* Object initialization

### Access Control

* Public access modifier
* Private access modifier
* Protected access modifier
* Package-level access

### Static Concepts

* Static variables
* Static methods
* Static blocks
* Inner classes

### Inheritance

* Single inheritance
* Multilevel inheritance
* Constructor chaining
* Method inheritance

### Polymorphism

* Method overloading
* Method overriding
* Runtime polymorphism

### Abstraction

* Abstract classes
* Abstract methods

### Interfaces

* Interface implementation
* Multiple inheritance using interfaces
* Nested interfaces
* Interface extension

### Packages

* Creating custom packages
* Importing packages
* Package organization

### Exception Handling

* Try-Catch blocks
* Custom Exceptions
* Exception propagation

### Generics

* Generic classes
* Generic methods
* Custom generic collections

### Multithreading

* Creating threads
* Thread lifecycle
* Thread priorities

### Design Patterns

* Singleton Pattern

### Object Cloning

* Shallow Copy
* Deep Copy
